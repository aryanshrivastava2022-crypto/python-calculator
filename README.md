# Calculator
A simple calculator built using Python.

## Features
- Addition
- Subtraction
- Multiplication
- Division

## Requirements
- Python 3.x

## How to Run
1. Clone the repository:
   git clone https://github.com/your-username/day01-calculator.git

2. Navigate to the project folder:
   cd day01-calculator

3. Run the program:
   python calculator.py
